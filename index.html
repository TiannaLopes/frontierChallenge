Problem 6

Below is a number input, a button, and a box. Using JavaScript, develop a solution that will translate whatever number is typed into the input into a Roman numeral representation and output this to the box, when the button is clicked. Use the following rules in interpreting the numbers:

  • Only positive numbers less than four thousand are allowed as input.
  • Use subtractive notation (e.g. IV instead of IIII, XC instead of LXXXX).
  • When a number cannot be parsed, either because it contains invalid characters or because it falls outside the allowed range, the text should read "Invalid input."

Some examples:

  • 1 → I
  • 4 → IV
  • 1984 → MCMLXXXIV
  • 4000 → Invalid input
  • 3999 → MMMCMXCIX
  • 0 → Invalid input
  • I like turtles → Invalid input
